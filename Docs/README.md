# Ecommerce  Backend

## Overview
Multi-vendor ecommerce system supporting:
- Admin
- Vendor
- Customer

## Tech Stack
- Node.js
- Express
- PostgreSQL
- JWT Auth
- OpenAPI 3.1
- Zod (Validation)

## Documentation Rules
We document:
- Features
- Database structure (collections + indexes)
- API contracts
- Architecture decisions
- Important bugs / lessons

We do NOT document:
- Internal helper functions
- Minor refactors
- Variable renames

Code explains HOW.
Docs explain WHAT and WHY.
